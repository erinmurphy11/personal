# erinmurphy.dev

Personal portfolio and blog, built with [Astro](https://astro.build), Tailwind, and TypeScript on the [Astro Nano](https://github.com/markhorn-dev/astro-nano) theme. Deployed on [Vercel](https://vercel.com).

## Commands

All commands are run from the root of the project, from a terminal:

| Command                   | Action                                            |
| :------------------------ | :------------------------------------------------ |
| `npm install`              | Installs dependencies                             |
| `npm run dev`              | Starts local dev server at `localhost:4321`       |
| `npm run dev:network`      | Starts local dev server on local network          |
| `npm run build`            | Build the production site to `./dist/`            |
| `npm run preview`          | Preview the build locally, before deploying       |
| `npm run preview:network`  | Preview the build on local network                 |
| `npm run astro ...`        | Run CLI commands like `astro add`, `astro check`  |
| `npm run lint`             | Run ESLint                                        |
| `npm run lint:fix`         | Auto-fix ESLint issues                            |
